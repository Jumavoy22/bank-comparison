# Дэшборд для сравнения банков

Этот Streamlit-дэшборд предназначен для анализа и сравнения банков по различным метрикам на основе данных о транзакциях.

## Запуск приложения

1.  Установите необходимые библиотеки:
    ```bash
    pip install streamlit pandas numpy
    ```
2.  Запустите приложение:
    ```bash
    streamlit run streamlit_app.py
    ```

## Описание метрик

Дэшборд рассчитывает и визуализирует следующие метрики для каждого банка:

-   **Общее количество транзакций (total_tx)**
    -   **Описание:** Общее число всех транзакций, проведенных по картам банка.
    -   **Чем выше, тем лучше.**

-   **Общая сумма транзакций (total_vol)**
    -   **Описание:** Общий объем денежных средств, прошедших через все транзакции банка.
    -   **Чем выше, тем лучше.**

-   **Средний чек (avg_check)**
    -   **Описание:** Средняя сумма одной транзакции (`total_vol / total_tx`). Показывает среднюю покупательскую способность или размер трат клиентов.
    -   **Чем выше, тем лучше.**

-   **Медианный чек (median_check)**
    -   **Описание:** Медианное значение суммы транзакции. Эта метрика менее чувствительна к выбросам (очень крупным или мелким транзакциям), чем средний чек, и лучше отражает "типичную" сумму покупки.
    -   **Чем выше, тем лучше.**

-   **95-й перцентиль суммы (p95)**
    -   **Описание:** Значение, ниже которого находятся 95% всех сумм транзакций. Эта метрика помогает понять порог для крупных, но не экстремальных трат.
    -   **Чем выше, тем лучше.**

-   **Доля крупных транзакций (high_value_share)**
    -   **Описание:** Процент транзакций, сумма которых превышает 95-й перцентиль, рассчитанный по всему рынку (всем банкам). Показывает, как часто клиенты банка совершают крупные покупки по сравнению с рынком в целом.
    -   **Чем выше, тем лучше.**

-   **Доля P2P-переводов (p2p_share)**
    -   **Описание:** Процент транзакций, которые являются переводами от одного физического лица другому (person-to-person).
    -   **Зависит от стратегии банка.**

-   **Средний чек P2P-перевода (p2p_avg)**
    -   **Описание:** Средняя сумма одного P2P-перевода.
    -   **Чем выше, тем лучше.**

-   **Доля топ-категории (top_cat_share)**
    -   **Описание:** Доля самой популярной категории транзакций (например, "супермаркеты", "рестораны") в общем количестве транзакций банка. Показывает, насколько сфокусированы траты клиентов.
    -   **Зависит от стратегии банка.**

-   **Доля топ-способа выдачи (top_issue_method_share)**
    -   **Описание:** Доля самого популярного способа выдачи карты (например, "в отделении", "онлайн") в общем количестве транзакций.
    -   **Зависит от стратегии банка.**

-   **Доля микро-транзакций (micro_share)**
    -   **Описание:** Процент транзакций, сумма которых составляет менее 10% от медианного чека по рынку. Показывает долю очень мелких покупок.
    -   **Чем ниже, тем лучше.**

-   **Коэффициент Джини (gini)**
    -   **Описание:** Метрика, показывающая степень неравенства в распределении сумм транзакций. Значение от 0 до 1.
        -   **0** означает абсолютное равенство (все транзакции на одну и ту же сумму).
        -   **1** означает абсолютное неравенство (одна транзакция на всю сумму, остальные - 0).
    -   Более низкий коэффициент Джини указывает на более равномерное распределение трат клиентов.
    -   **Чем ниже, тем лучше.**
